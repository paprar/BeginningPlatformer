현재 오브젝트의 아래쪽에 정렬된 오브젝트에 HealthHeart 컴포넌트가 부착되있고 Image-sprite가 empty 상태일때 Image-sprite를 변경할 수 있도록 하기.

하이어라키창에서 찾아서 어떤 상태인지 검사하고 적용하고 상태를 바꿔보려니까 스택 오버플로를 발생시켰다는 오류가 뜸.

StackOverflowException: The requested operation caused a stack overflow


-> 부모 오브젝트에 새 스크립트 달아서 다음 자식 오브젝트에 HealthHeart 컴포넌트가 부착되있고 empty 상태일때 이전 오브젝트의 상태를 변경할 수 있게끔 해보니까 오류는 안뜸.

현재는 오브젝트의 상태를 바꿀때 부모 오브젝트의 막내 오브젝트부터 바꾸고 이게 empty 상태일때 다른 오브젝트의 상태도 바꿀 수 있도록 했는데 막내 오브젝트의 상태만 바꿔짐.

아니면 다른 수식을 활용한 더 간단한 방식이 있는지 궁금함.


r키를 눌렀을때 부모 오브젝트의 마지막 자식 오브젝트의 상태를 full에서 half로 바꾸고 다시 한번 누르면 empty로 바꾸고 또 한번 누르면 마지막 자식 오브젝트의 이전 자식 오브젝트의 상태를 full에서 half로 바꾸고 또다시 누르면 empty로 바꾸고 더 누르면 이전의 이전 자식 오브젝트의 상태를 변경하는걸 반복하게 함. 그랬더니 됨.

문제는 그걸 함수로 만들어서 충돌했을때 호출하게 했는데 호출이 안된건지 null exception 이런식으로 뜸. -> 다른 오브젝트에서 컴포넌트 불러오는 코드 쓰니까 됨.
